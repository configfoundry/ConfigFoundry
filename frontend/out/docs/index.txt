3:T12a6,<h1 id="configfoundry-documentation">ConfigFoundry Documentation<a href="#configfoundry-documentation" class="heading-anchor" aria-label="Link to this section">#</a></h1>
<p>ConfigFoundry is a shared SNMP/ICMP collector configuration generator: a single source of truth for network device inventory, bandwidth caps, and subnet definitions, with a web UI and REST API that generate collector YAML configs on demand. It ships with enterprise authentication (JWT + MFA + API keys), fine-grained RBAC, an IP-based Access Policy Engine, a storage abstraction that runs on SQLite or PostgreSQL/MySQL/SQL Server, and a fully air-gap-capable deployment path.</p>
