## Top priority: port the Network Tree to the Next.js frontend

The pan/zoom spatial device diagram exists only in the original vanilla-JS frontend (`static/`), fully superseded once `frontend/out/` is present, and the legacy frontend predates the authentication layer so it can't be exposed as a working fallback. This is the one feature genuinely unavailable in the current frontend, not merely undocumented. See [[Features/Feature - Network Tree|Feature - Network Tree]].
